# 事件关联公司挖掘（Task 2）

## 四维关联评分体系

| 维度 | 权重 | 说明 |
|---|---|---|
| `direct_mention` | 0.45 | 新闻文本直接提及公司名称 |
| `business_match` | 0.25 | 主营业务关键词与事件匹配 |
| `industry_overlap` | 0.20 | 事件行业与股票行业重合度 |
| `historical_co_move` | 0.10 | 历史价格共振强度 |

关联强度 = Σ(维度权重 × 维度得分)，低于 0.2 的关系被过滤。

当前基础权重由 `config/config.yaml` 中的 `scoring.association` 控制；若需要按事件驱动主体类型做差异化调整，则通过 `scoring.association_profiles` 配置倍率后再归一化。

这次修复前，配置文件里的关联权重和实际代码行为并不一致；现在已经收敛成真正的配置驱动口径。

## 产业链映射

`data/manual/industry_relation_map.json` 定义了五类主题的完整产业链：

- **军工**：整机制造 → 机载配套 → 导弹弹药 → 军工材料
- **科技/算力**：算力基础设施 → 高速互联 → 半导体设备 → 高端芯片
- **新能源**：逆变器储能 → 光伏材料 → 新能源车电池
- **低空经济**：通信基础设施 → 飞行器配套
- **业绩预告**：盈利兑现 → 成长驱动 → 景气兑现

每个主题包含具体的关联股票列表（如中航沈飞、晨曦航空、中际旭创等）。
